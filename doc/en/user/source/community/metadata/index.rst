.. _community_metadata:

Metadata
========

The ``Metadata module`` adds a metatada tab to the layer configuration allowing the user to have the layer configuration and layer metadata in one central place.
The layer metadata can be harvested from geoserver by `GeoNetwork <https://geonetwork-opensource.org//>`_ .

The content of the metadata tab is fully customizable see :ref:`community_metadata_configuration`
